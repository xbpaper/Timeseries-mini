## 第一步：前端 React 核心页面

你只需要做 4 个组件，非常简单：

### 1. 用户数据上传CSV格式 & 预览

```jsx
<input type="file" accept=".csv" onChange={handleUpload} />
<table>{previewData}</table>
```

### 2. 数据清洗面板（按钮点选）

- 缺失值填充：均值/前向填充
- 异常值剔除：3σ 原则
- 时间重采样：1min / 5min / 1h
- 字段筛选：勾选保留哪些测点

全部做成 **checkbox + select**，后端用 Pandas 执行。

### 3. 训练面板

```jsx
<button onClick={() => api.train(params)}>开始训练</button>
<ProgressBar />
<Chart显示损失曲线 />
```

### 4. 微调和生成智能体

- 上传新数据 → 点击“微调”
- 微调完成 → 生成一个推理接口 URL
- 这个 URL 就是你的**工业智能体**，可以给 SCADA、大屏调用

## 第二步：后端核心

我直接给你**最简可用代码结构**，不用懂深度学习。

### 1. 安装依赖

```txt
pandas numpy fastapi uvicorn
scikit-learn torch pytorch-lightning
tsai # 时序模型库
```

### 2. 数据清洗接口（核心就这点）

```python
@app.post("/clean")
def clean_data(file: UploadFile):
    df = pd.read_csv(file.file)
    df = df.drop_duplicates()
    df = df.fillna(method="ffill")  # 前向填充
    # 异常值剔除
    for col in cols:
        mean = df[col].mean()
        std = df[col].std()
        df = df[(df[col] > mean-3*std) & (df[col] < mean+3*std)]
    return df.to_json()
```

### 3. 训练时序模型（用现成轻量模型）

```python
from tsai.models.TimesNet import TimesNet

@app.post("/train")
def train(data_json):
    df = pd.read_json(data_json)
    X, y = create_rolling_windows(df)  # 滑窗构造时序样本
    model = TimesNet(c_in=X.shape[1])
    train(model, X, y)
    save_model("time_series_model.pt")
    return {"loss": 0.05, "status": "success"}
```

### 4. 微调（增量训练 = 极简版 LoRA）

```python
@app.post("/finetune")
def finetune(new_data):
    model = load("time_series_model.pt")
    new_X, new_y = prepare_data(new_data)
    train(model, new_X, new_y, lr=1e-5)  # 小学习率=微调
    save_model("finetuned_model.pt")
    return {"msg": "微调完成"}
```

### 5. 推理接口 = 你的工业智能体

```python
@app.post("/agent/predict")
def agent_predict(input_data):
    out = model(input_data)
    return {"预测值": out.tolist()}
```
